# 🌸 Petalalyze — розпізнавання квітів у букеті

**Petalalyze** — мобільний застосунок для автоматичного розпізнавання квітів на фотографіях букетів.  
Система поєднує **детекцію об’єктів** та **класифікацію зображень**, щоб знаходити окремі квітки на фото, визначати їхній вид і зберігати результати для подальшого перегляду.

---

## ✨ Основні можливості

### 🔍 Розпізнавання квітів

- завантаження фотографії букета;
- автоматична **детекція квіток** на зображенні;
- вирізання знайдених областей;
- класифікація кожної знайденої квітки.

### 📊 Відображення результатів

- відображення **bounding boxes** на вихідному зображенні;
- показ передбаченого класу та рівня впевненості;
- збереження результату розпізнавання в історію;
- присвоєння кожному запису унікального `detection_id`.

### 🕘 Історія розпізнавань

- перегляд списку попередніх розпізнавань;
- відкриття деталей конкретного результату;
- видалення записів з історії.

### 🌼 Каталог квітів

- список доступних квітів;
- короткі описи для ознайомлення з видами.

---

## 🧠 Як працює система

Обробка зображення складається з кількох етапів:

1. Користувач робить або завантажує фото букета у мобільному застосунку.
2. Фото передається на **FastAPI backend**.
3. Модель **YOLO** знаходить на зображенні окремі квітки.
4. Для кожної знайденої області створюється crop.
5. Кожен crop класифікується ансамблем **Vision Transformer (ViT)** та **ConvNeXt**.
6. Результати повертаються у Flutter-застосунок.
7. Дані відображаються на екрані та зберігаються в базі даних.

---

## 🏗 Архітектура проєкту

Проєкт складається з двох основних частин:

- **Backend** — FastAPI-сервер, який відповідає за ML-обробку, API та роботу з базою даних.
- **Frontend** — мобільний застосунок на Flutter для взаємодії користувача із системою.

---

## 🛠 Технології

### Backend

- **FastAPI**
- **SQLAlchemy**
- **Uvicorn**
- **YOLO** — для детекції квіток
- **ViT + ConvNeXt** — для класифікації видів квітів

### Frontend

- **Flutter**
- **Dart**

├── backend/                 # FastAPI + ML

│   │   ├── main.py          # точка входу, static mounts, lifespan

│   └── README.md

## 🚀 Запуск бекенду

```powershell
cd backend
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

## 📱 Запуск Flutter
